📋 Customer Transaction History Feature

🎯 New Feature Added:

When you select a customer, their complete transaction history appears with:

📋 Transaction Display Shows:

Transaction ID: #1234
Date: 2024-01-15
Type: Sale
Status: Remaining
Total Amount: 500.00 DH
Advance Paid: 200.00 DH
Remaining: 300.00 DH
Action:

📊 Customer Summary:

Total Advance: 850.00 DH Total Remaining: 1,200.00 DH

🛒 How It Works:

  • Select Customer: Transaction history automatically loads
  • View Details: Each transaction shows ID, total, advance, remaining
  • Pay Rest: Click "Pay Rest" button to pay remaining amount
  • Real-time Update: Transactions update immediately after payment
  • Summary: Shows total advance and remaining across all transactions

🔧 API Endpoints Created:

  • GET /api/customers/transactions.php?client_id=X - Get customer transactions
  • POST /api/customers/pay_remaining.php - Pay remaining balance
  • Database Tables: sales, customer_payments
